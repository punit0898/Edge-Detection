# Real-Time Edge Detection Viewer

An Android application that captures camera frames, processes them using OpenCV C++ for edge detection, renders the output with OpenGL ES, and includes a TypeScript web viewer for displaying processed frames.

## Features

### Android App
- Real-time camera feed capture using Camera2 API
- Edge detection processing using OpenCV C++ via JNI
- OpenGL ES 2.0 rendering for smooth performance
- Toggle between original and edge-detected views
- FPS counter display
- Modular architecture with proper separation of concerns

### Web Viewer
- TypeScript-based web interface
- Display of processed frames with statistics
- Toggle invert display mode
- Sample frame loading
- Real-time data simulation
